# Changelog

## v2.0.0 - 2026-02-20

- 修复“怎么答都同结果”问题：
  - 从固定模板改为 **18题多维加权评分引擎**
  - 角色由综合分动态计算（DPS/Tank/Support）
  - 风格由维度自动判定（flanker/poke/brawl/flex）
- 英雄资源改为真实数据：
  - 前端优先实时拉取 OverFast API（quickplay 英雄）
  - 网络失败时回退本地快照 `data/heroes-live.json`
  - 英雄卡片使用真实英雄 portrait（不再笑脸占位）
- 产品化增强：
  - 结果包含主定位 + 副定位 + 风格 + 主副英雄池 + 建议
  - 雷达图保留，分享海报（3:4）继续可用
  - 文案声明数据来源与回退机制
