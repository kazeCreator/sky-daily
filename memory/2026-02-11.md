## Bird Skill (X/Twitter CLI) - 只读模式已配置

### 安装方式
- 从 GitHub openclaw/skills 仓库手动 clone 到 `/tmp/openclaw-skills`
- 复制到 `~/.openclaw/workspace/skills/bird`

### 只读模式配置
- 原 binary: `/opt/homebrew/bin/bird.real`
- wrapper script: `/opt/homebrew/bin/bird` (拦截写操作)
- 拦截的命令: `tweet`, `reply`, `follow`, `unfollow`, `unbookmark`
- 允许的命令: `read`, `thread`, `search`, `user-tweets`, `mentions`, `whoami`, `check`

### 恢复可写
```bash
mv /opt/homebrew/bin/bird.real /opt/homebrew/bin/bird
```
